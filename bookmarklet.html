Note: Your bookmarks bar must be visible. In Chrome/Edge press Ctrl+Shift+B to show it.

The bookmark stores your GitHub token in this site's localStorage — it is never sent anywhere except the GitHub API.

Step 2 — Use it

  1. Go to dramexchange.com and let the page fully load.
  2. Click the 📊 Update DRAM Prices bookmark.
  3. If prompted, enter your GitHub token (only needed once per browser).
  4. A confirm dialog will ask about 512Gb TLC — click OK only if it updated this week, otherwise Cancel.
  5. Done — the live dashboard updates within ~30 seconds.
